## Data-Miner

The project aims to plot the following algorithms into different visualisation formats like Live-Graphs, Diagrams, Bar-plots, etc.

1. Regression.
    * Linear Regression.
  
2. Classification.
    * Decision Tree 
    * SVM.
    * PCA.
    * KNN Classifier.
  
3. Clustering. 
    * Fuzzy C-Means
    * KMeans 
    * Hierarchical Clustering 



## Getting Started

* Fork the repository.

* npm install. 

* npm start.

* Open your localhost(XXXX) on your Web-browser.
